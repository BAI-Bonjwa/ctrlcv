### 快捷键

| 快捷键    | 说明              |
| --------- | ----------------- |
| `h`       | 显示帮助          |
| `q`       | 退出程序          |
| `m`       | 切换成内存视图    |
| `k`       | 杀死进程          |
| `Shift+p` | 按 CPU 使用率排序 |
| `Shift+m` | 按内存使用率排序  |
| `Shift+r` | 反转排序          |
| `Shift+l` | 搜索字符串        |
| `o`       | 添加一个过滤器    |
| `=`       | 清楚全部过滤器    |
