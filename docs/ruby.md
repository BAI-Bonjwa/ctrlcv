### Reference

| Code                    | Description                                 |
| ----------------------- | ------------------------------------------- |
| `$!`                    | 最新的错误信息                              |
| `$@`                    | 错误的位置                                  |
| `$_`                    | gets 方法得到的最后一个字符串               |
| `$.`                    | irb 中最后一个字符的行号                    |
| `$&`                    | 正则表达式的最后一个匹配的字符串            |
| `$~`                    | 在当前的 Scope 中，匹配最新的一个正则表达式 |
| `$n`                    | 在最后的匹配中，第 n 个子表达式             |
| `$=`                    | 大小写敏感                                  |
| `$/`                    | 输入记录分隔符                              |
| `$\`                    | 输出记录分隔符                              |
| `$0`                    | Ruby 脚本文件的名称                         |
| `$*` (or `ARGV`)        | 命令行参数                                  |
| `$$`                    | 解释器的进程 ID                             |
| `$?`                    | 最后一个子进程的退出状态                    |
| `$-i` `$-l` `$-p` `$-v` | 命令行切换                                  |
| `$-v` (or `$VERBOSE`)   | 调试模式                                    |
